# python-backend-challenges
Тестовые задания для собеседований

Задание:
Написать скрипт-замерятель скорости интернета со своего компьютера.
Он должен принимать адрес, куда стучаться (какая-нибудь тяжелая картинка), 
запускать последовательно 10 запросов к этому адресу, дожидаться ответа, 
вычислять среднее время запроса, объем скачанных данных и печатать в консоли скорость мб/с.

# Download Speed Test

Консольная программа на Python для измерения скорости загрузки изображения.

Программа принимает URL изображения, последовательно загружает его 10 раз и выводит:

* среднее время выполнения одного запроса;
* общий объём загруженных данных;
* среднюю скорость загрузки в МБ/с.

## Требования

* Python 3.13+
* pip

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/Grepka/python-backend-challenges.git
cd python-backend-challenges
```

Создайте виртуальное окружение:

```bash
python3 -m venv .venv
```

Активируйте виртуальное окружение на macOS или Linux:

```bash
source .venv/bin/activate
```

На Windows:

```powershell
.venv\Scripts\activate
```

Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

## Запуск

Передайте URL изображения первым аргументом командной строки:

```bash
python speedtest.py https://example.com/image.jpg
```

Для просмотра справки:

```bash
python speedtest.py --help
```

## Пример результата

```text
2026-06-26 14:10:01 | INFO | Загрузка 1/10 завершена: 0.245 сек., 8991 байт
2026-06-26 14:10:02 | INFO | Загрузка 2/10 завершена: 0.137 сек., 8991 байт
...

Итоги:
Среднее время запроса: 0.152 сек.
Объём скачанных данных: 0.090 МБ
Средняя скорость: 0.059 МБ/с
```

## Особенности реализации

Запросы выполняются последовательно с использованием одной HTTP-сессии.

Тело ответа считывается частями, поэтому изображение не загружается целиком в оперативную память.

Для измерения длительности используется `time.perf_counter()`.

Средняя скорость рассчитывается как отношение общего объёма загруженных данных к суммарному времени выполнения запросов:

```text
средняя скорость = общий объём данных / общее время загрузки
```

Один мегабайт считается равным `1 000 000` байт.

При ошибке HTTP-запроса или превышении тайм-аута программа завершается с ненулевым кодом возврата.
